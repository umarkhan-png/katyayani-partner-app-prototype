# Deploy Katyayani Partner App Prototype

## Quick Deploy (Netlify Drop — 2 minutes)

1. Open https://app.netlify.com/drop in your browser
2. Drag the entire `katyayani-partner-app` folder into the drop zone
3. Wait 30 seconds — you'll get a public URL like `https://random-name-12345.netlify.app`
4. Share that URL with anyone — works on any device, any browser

## Optional: Custom URL

After the first deploy, click **Site settings → Change site name** to set a custom subdomain like `katyayani-partner-app.netlify.app`.

## Update the prototype later

Just drag-drop the folder again on the same site. Netlify will replace files and keep the same URL.

## Need a permanent URL?

Use **GitHub Pages** instead:
1. Create a free GitHub account
2. Create a new repo (e.g. `katyayani-partner-app`)
3. Upload all files
4. Go to Settings → Pages → Source: `main` branch
5. Get a URL like `https://yourusername.github.io/katyayani-partner-app/`
