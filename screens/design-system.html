08 · Spacing & Border Radius
Border Radius Scale (consolidated)
10px — rounded-[10px]
CANONICAL — cards, tiles, buttons, sections, inputs (everything)
999px — rounded-full
Pills, nav dots, avatars (circular only)
24px top — bottom sheets only
rounded-t-[24px] — drag-up modals
Removed: rounded-md (6) · rounded-lg (8) · rounded-xl (12) · rounded-2xl (16) · rounded-3xl. All map → rounded-[10px].
Spacing Scale (px)
4px   — gap-1   · micro gap between icons
8px   — gap-2   · icon + label, tight pairs
12px  — gap-3   · card internal padding
16px  — p-4     · standard card padding
20px  — p-5     · section padding mobile
24px  — p-6     · section padding desktop
32px  — p-8     · page outer padding
48px  — gap-12  · major section separation

Screen padding: px-4 (16px) on mobile
Card inner:     p-4 (16px) standard
Section gap:    space-y-4 (16px) between cards

─────────────────────────────────────────────
rounded-[10px]        →  BorderRadius.circular(10)
rounded-t-[24px]      →  BorderRadius.vertical(top: 24)
border 1.5px #0E7A4E  →  Border.all(width: 1.5, color: AppColors.green)
font-weight: 700      →  FontWeight.w700
letter-spacing: -0.02 →  letterSpacing: -0.32 (px ≈ size × ls)
box-shadow            →  BoxShadow(blurRadius:, offset:, color:)
backdrop-filter       →  BackdropFilter + ImageFilter.blur
linear-gradient       →  LinearGradient(begin:, end:, colors:[])
animation: spin       →  AnimationController + RotationTransition
position: absolute    →  Positioned (in Stack)
bottom: env(safe-area)→  MediaQuery.of(c).padding.bottom
overflow: hidden      →  ClipRRect

                       muted text #9E9E9E only on labels, never running text
