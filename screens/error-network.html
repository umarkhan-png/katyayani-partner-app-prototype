Implementation notes:
• Offline: detect via navigator.onLine — show full-screen block
• Slow: show toast when request > 3s, dismiss automatically at 8s
• 500/503: show with request ID for support escalation
• Timeout: 15s threshold; auto-retry once before showing error
• All errors must offer a Retry button + option to go Home
• Cache last-known prices/cart in SQLite for offline viewing
