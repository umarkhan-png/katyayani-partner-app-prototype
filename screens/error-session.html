Implementation notes:
• Session expiry: JWT token expiry = 30 min inactivity / 7 days max
• Preserve cart and form state in local storage across session expiry
• Show bottom sheet (not full-screen redirect) for expired sessions
• Account suspended: always provide direct contact (KAM + WhatsApp)
• New device alert: send push + in-app; offer 1-tap "Secure account"
• Never show error codes to users — use friendly messages
